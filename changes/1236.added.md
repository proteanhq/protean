Added a typed value-object dictionary field: `Dict(value_type=ValueObject(SomeVO))` declares a `dict[str, SomeVO]`, a string-keyed `code -> value object` map. Each value is reconstructed and validated as the value object on load and serialized back to a plain dict, persisted as a single JSON column, mirroring `List(content_type=ValueObject(...))`. The untyped `Dict()` is unchanged for loose JSON.
