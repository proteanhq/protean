Fix `domain.create_snapshots(Aggregate)` failing for event-sourced aggregates declared with `fact_events=True`. The bulk snapshot API mistook the aggregate's `{category}-fact-{id}` fact-event streams for real instances and raised `IncorrectUsageError` (fact events have no `@apply` handler). `create_snapshots` now skips those fact-event streams for fact-event-enabled aggregates.
