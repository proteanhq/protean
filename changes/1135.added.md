`protean events catalog` lists every concrete event (those belonging to an aggregate cluster) with its version, deprecation and supersession status, upcaster chain, and consumers (event handlers / projectors / process managers), sourced from the domain IR. Renders a human table or `--json`, and works from a live domain (`--domain`) or a serialized IR file (`--ir`). `protean schema generate --format all` emits the matching versioned `.protean/schemas/` tree in every format (JSON + Avro + Protobuf) in one pass — together they form the local schema-registry on-ramp.
