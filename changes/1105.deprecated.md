The `Method` and `Nested` serializer fields, and the `pickled` argument on `List`, are deprecated and will be removed in v1.0.0. `Method`/`Nested` supported a serializer element that no longer exists, and `List(pickled=...)` was a dead kwarg with no effect. Each surfaces a `DeprecationWarning` (`RemovedInProtean10Warning`) at field-declaration time; run Python with `-W default::DeprecationWarning` to see them. `ValueObjectList` is **not** deprecated (it remains live framework machinery), but at the user declaration surface prefer `List(content_type=ValueObject(...))`.
