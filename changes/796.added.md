Add a development hot-reload mode to `protean server`. Pass `--reload` to watch the working directory for Python source changes and automatically restart the inner Engine process on every edit. Built on `watchfiles` (installed via the new `protean[dev]` extra); `.pyc`, `__pycache__`, `.venv`, `.git`, `node_modules`, and `.protean/` caches are ignored. `--reload` cannot be combined with `--workers > 1`.
