Adopt OpenTelemetry 1.43 (`opentelemetry-api`/`opentelemetry-sdk`/`opentelemetry-exporter-otlp-proto-grpc` >= 1.43.0, `opentelemetry-instrumentation-fastapi`/`opentelemetry-exporter-prometheus` >= 0.64b0). `TraceParent` now masks the W3C trace-flags byte (`0x01`) instead of comparing it literally, so headers carrying the new `RANDOM_TRACE_ID` flag (e.g. `03`) are read as sampled.
