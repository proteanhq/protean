Emit `protean.adapters.repository.sqlalchemy.slow_query` WARNING events when a SQLAlchemy query exceeds the configured threshold, and `protean.adapters.repository.sqlalchemy.query` DEBUG events for every query (opt-in via level configuration). Events carry `statement`, `parameters`, `duration_ms`, and `threshold_ms`, are truncated via `[logging].slow_query_truncate_chars` (default 500), and auto-gain `correlation_id` / `causation_id` through the root logger's `ProteanCorrelationFilter`. Threshold comes from `[logging].slow_query_threshold_ms` (default 100ms).
