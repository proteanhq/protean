Removed two CLI surfaces as part of the 1.0 API consolidation:

- The `protean generate docker-compose` command (and the now-empty `generate`
  command group). It was a no-op stub that wrote no file; real Dockerfile/compose
  generation is tracked separately in #397.
- The deprecated `--debug` flag on `protean server` and `protean observatory`,
  which completed its 0.16 → 0.17 deprecation cycle. Use the global
  `--log-level DEBUG` flag instead (for example, `protean --log-level DEBUG server`).

Also marked `protean test` and `protean test test-adapter` as
`[Framework development]` tooling in their `--help` output, since they drive
Protean's own pytest suite and are not intended for end-user applications.
