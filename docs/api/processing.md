# Processing priorities

Priority-based event routing for production and backfill lanes. Events can
be routed to different processing streams based on priority, enabling
bulk migrations without disrupting real-time event processing.

See [Priority Lanes](../concepts/async-processing/priority-lanes.md) for
the conceptual explanation.
