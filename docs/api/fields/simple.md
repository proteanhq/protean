# Simple fields

Factory functions that return `FieldSpec` instances for common scalar types.
Each factory accepts keyword arguments for validation constraints like
`required`, `max_length`, `min_value`, `choices`, etc.

See [Simple Fields reference](../../reference/fields/simple-fields.md) for
a summary table and [Field Arguments](../../reference/fields/arguments.md)
for all available keyword arguments.
